# Manuel d'utilisation

## Presentation

Abiotic Crafting Notes (ACN) est un carnet de commandes collaboratif pour le jeu Abiotic Factor. Il permet aux joueurs de:

- Commander des items a crafter
- Voir et accepter des commandes
- Indiquer les ressources manquantes
- Rechercher des recettes
- Calculer les ressources necessaires

## Pages

### Commandes

Page principale pour gerer les commandes de crafting.

**Creer une commande:**
1. Cliquer sur "Nouvelle commande"
2. Rechercher les items souhaites
3. Ajuster les quantites avec +/-
4. Ajouter des notes si necessaire
5. Cliquer sur "Envoyer la commande"

**Accepter une commande:**
1. Voir les commandes "En attente"
2. Cliquer sur "Accepter"
3. La commande vous est assignee

**Terminer une commande:**
1. Voir les commandes "En cours"
2. Cliquer sur "Terminer" quand c'est fait

**Indiquer des ressources manquantes:**
1. Sur une commande acceptee, cliquer sur "Ressources manquantes"
2. Indiquer les items et quantites manquants

### Recettes

Page pour explorer les recettes du jeu.

- Rechercher par nom
- Filtrer par categorie
- Voir les details d'une recette (ingredients, poids, durabilite)
- Voir l'arbre de dependances
- Voir le total des ressources de base necessaires

### Calculateur

Page pour calculer les ressources totales pour plusieurs items.

1. Selectionner les items et quantites
2. Cliquer sur "Calculer les ressources"
3. Voir le total agrege des ressources de base

## Statuts des commandes

| Statut | Description |
|--------|-------------|
| En attente | Commande creee, en attente d'un crafter |
| Acceptee | Un crafter a pris la commande |
| En cours | Le crafting est en cours |
| Ressources manquantes | Le crafter a besoin de materiaux |
| Terminee | Commande livree |
| Annulee | Commande annulee par le demandeur |
